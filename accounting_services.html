Услуги по бухгалтерскому и налоговому учету

Главная / Услуги / Услуги по бухгалтерскому и налоговому учету

Услуги

  • Ведение учета в соответствии с учетной политикой предприятия по следующим процессам:
    • - учёт операций с ТМЦ
    • - учёт банковских операций
    • - учёт финансовых вложений
    • - учёт кредитов и займов
    • - учёт операций с внеоборотными активами
    • - учёт операций по приобретению и оказанию работ и услуг
    • - учёт налоговых обязательств
    • - формирование себестоимости
    • - закрытие периода, формирование индивидуальной отчетности и заполнение ФСД
    • - учёт операций по реализации продукции/товаров
    • - учёт операций по приобретению товаров
    • - учёт расчетов с персоналом по оплате труда
    • - учёт операций с капиталом
  • Предоставление оперативных справок, данных для формирования консолидированной отчётности
  • Оказание содействия при формулировании позиции в налоговых спорах
  • Выполнение контрольных процедур, находящихся в зоне ответственности ЦО
  • Проведение выверки ВГО
  • Функция главного бухгалтера
